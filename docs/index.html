Recuerda que he propuesto este recorrido:

  • Escribe el texto en nano.
  • Utiliza sintaxis Markdown.
  • Una vez que tengas el texto, transfórmalo en HTML:
    • De forma manual.
    • Copia el Markdown en la página de prueba de Pandoc y conviértelo desde ese menú (parse).
    • Descarga de Pandoc.org.
    • Instala con chocolatey.
  • Copia ese HTML y pégalo en el código de esta parte del contenido.

Y eso no es todo, tenemos que verlo ofrecido por un servidor web:

  • En nuestro ordenador o localhost con python, python3 -m http.server --bind 127.0.0.1 1312.
  • En la Web con Github.
